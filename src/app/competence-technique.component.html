Développer une application de test avec Typescript

Une application de test permet de tester le bon fonctionnement d'une application ou bien d'une API.

Elle est en général écrite en Typescript ou en Javascript.

Typescript est un langage créé Microsoft. C'est un sur-ensemble de Javascript. Ce langage, Typescript, a été créé afin de faciliter les bonnes pratiques de programmation en Javascript, et de fournir ainsi un code en arrière-plan de qualité.
